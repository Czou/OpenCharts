Summary
OpenCharts.Coordinate图表坐标轴风格
Properties
unit{String} 图表数据单位，默认为空
stepNumber{Number} Y轴段数，默认为10 stepRange为-1时，采用stepNumber方式对Y轴进行分段。 比如Y轴为15，分段数目为5，则每段数值为3 该属性目前不支持饼图等没有坐标轴的图表类型
stepRange{Number} Y轴每段数值，默认为-1 stepRange为-1时，采用stepNumber方式对Y轴进行分段，否则则以stepRange来分段 比如Y轴为15，每段数值为3，则会分为5段 该属性目前不支持饼图等没有坐标轴的图表类型
headRange{Number} 坐标轴顶端预留空白大小，像素单位
footerRange{Number} 坐标轴底部预留空白大小，像素单位
headRange{Number} 坐标轴顶端预留空白大小，像素单位
bShowAxes{Boolean} 是否显示坐标轴
bShowXAxesGrid{Boolean} 是否显示X坐标轴
bShowYAxesGrid{Boolean} 是否显示Y坐标轴
bShowXAxesText{Boolean} 是否显示X坐标轴文字
bShowYAxesText{Boolean} 是否显示Y坐标轴文字
axesStyle{OpenCharts.Style} 坐标轴风格
axesTextStyle{OpenCharts.TextStyle} 坐标轴文本风格
